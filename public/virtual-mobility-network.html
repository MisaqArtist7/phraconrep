Virtual Mobility Network

As one of COST Action Virtual Networking Tools, Virtual Mobility (VM) can help develop skills and capacities through virtual collaborations among researchers. Thus, you can apply for Virtual Mobility Grants (VMG) for networking that may facilitate reaching COST Action CA22115 “PhraConRep” deliverables. For details, please read COST Annotated Rules: https://www.cost.eu/funding/how-to-get-funding/documents-guidelines/.

The program introduces the following Virtual Networking Tools (VNT):

  • Virtual Networking Support (VNS) Grant: This grant focuses on enhancing virtual collaboration across all members of a specific COST Action. A designated Virtual Networking Support Manager, chosen by the COST Action’s Management Committee (MC), will spearhead this initiative. Responsibilities include developing a strategy for virtual networking, overseeing Virtual Mobility Grants, and assisting the MC in organizing and planning virtual events and collaborative activities.

  • Virtual Mobility (VM) Grant: Aimed at strengthening existing networks, this grant facilitates collaboration, knowledge exchange, and learning of new techniques among individual participants, without the need for physical presence. Selected by the MC, grantees engage in activities like conducting surveys, creating questionnaires, drafting protocols, and virtual mentoring. These efforts are geared towards building new capacities and skills.
